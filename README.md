# Africa-water-quality-testing-fellowship
The datasets included in this repository were collected to evaluate the Africa Water Quality Fellowship Program, described in the publication:

An experiential model for training the next generation of water quality professionals in sub-Saharan Africa: Lessons from a collaborative university fellowship program (currently in review)

The datasets include survey responses and quiz results from three cohorts of fellows participating in the program at universities in Ethiopia and Uganda (please see the publication for further details on the fellowship program and these datasets).
